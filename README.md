# Task Manager App

A beautiful, interactive task manager to stay organized and get things done — with a fun Boss Baby assistant!

**Live App:** https://vishnupreetha-gif.github.io/task-manager/

## How to Use

### Adding a Task
1. **Step 1 — Task Name:** Type what you need to do in the input field and click **Next**
2. **Step 2 — Due Date:** Pick a due date or click **Skip** if there's no deadline
3. **Step 3 — Category:** Choose a category — **Work**, **Personal**, **Shopping**, or **Study**
4. **Step 4 — Priority:** Select priority level — **Low**, **Medium**, or **High**
5. Click **Add Task** and your task appears in the list!

### Managing Tasks
- **Complete a task:** Click the checkbox next to a task to mark it as done
- **Edit a task:** Click the edit (pencil) icon to update the task name
- **Delete a task:** Click the delete (trash) icon to remove a task

### Search & Filter
- **Search:** Use the search bar to quickly find tasks by name
- **Filter by status:** Click **All**, **Pending**, or **Done** to filter tasks
- **Filter by category:** Click the category icons (Work, Personal, Shopping, Study) to view tasks from a specific category

### Progress Tracking
- The **progress bar** at the top shows how many tasks you've completed
- The **stats section** at the bottom shows your total, completed, and pending task counts

### Dark Mode
- Click the **moon icon** in the top-right corner to switch between light and dark mode

### Boss Baby Assistant
- The animated Boss Baby character reacts as you add and complete tasks — keep her happy by staying productive!

## Features
- Step-by-step task creation wizard
- 4 categories: Work, Personal, Shopping, Study
- 3 priority levels: Low, Medium, High
- Due date tracking
- Search and filter tasks
- Progress bar and stats
- Dark mode support
- Animated Boss Baby assistant
- Data saved in your browser (localStorage)
- Works on desktop and mobile
