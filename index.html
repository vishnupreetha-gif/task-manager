    How to Use Task Manager

    Adding a Task

    1. Step 1: Type your task name and click Next
    2. Step 2: Pick a due date or click Skip
    3. Step 3: Choose a category — Work, Personal, Shopping, or Study
    4. Step 4: Set priority — Low, Medium, or High
    5. Click Add Task and you're done!

    Managing Tasks

    • Click the checkbox to mark a task as done
    • Click the edit icon to rename a task
    • Click the delete icon to remove a task

    Search & Filter

    • Use the search bar to find tasks by name
    • Filter by All, Pending, or Done
    • Filter by category using the icons

    Extra Features

    • Dark Mode — click the moon icon to switch themes
    • Progress Bar — tracks your completion percentage
    • Boss Baby — she reacts as you add and complete tasks!
